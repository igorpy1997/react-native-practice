# 🐱 Cat Browser

A React Native app built with Expo to browse and favorite cats.

## Features

- 🐾 Browse a list of random cats
- ❤️ Add cats to favorites
- 📱 Bottom tab navigation
- 🗂️ Drawer menu
- 📄 Cat detail screen

## Tech Stack

- React Native
- Expo
- React Navigation (Stack, Bottom Tabs, Drawer)
- React Context API
